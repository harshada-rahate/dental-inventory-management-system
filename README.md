# dental-inventory-management-system-

A full-stack web application for managing dental inventory, including grafts, membranes, suppliers, and stock tracking.

## Tech Stack
- Java
- Spring Boot
- MySQL
- React.js
- HTML
- CSS
- JavaScript

## Features
- Manage Grafts
- Manage Membranes
- Manage Suppliers
- CRUD Operations
- REST APIs
- Inventory Tracking
